About Mel.

Who Am I?

My name is Melissa and I am a fourth year School of Interactive Art and Technology student at Simon Fraser University. Graphic design has always been an interest of mine as I was always fascinated by how visuals enhanced the way people communicate. After entering the program, I immediately fell in love with user experience (UX) as well as user interface (UI) design. Aside from being able to strengthen communication through design, UX/UI allows me to challenge myself with the complexity of user testing in design. Following my interest, my goal is to explore how design could improve everyday life in hopes of creating my own application one day.

Interactive Elements

In-text link

CSS selector:

.intext-link

Sample code:

<a href = "css/main.css" class ="in-text-link"> this is an in-text link </a>

Rendered element:

this is an in-text link

Navigation Link

CSS selector:

.view-details

Sample code:

<a class="view-details" href="https://www.google.com/">Detailst</a>/code>

Rendered element:

Details

Submit/Reset Buttons

CSS selector:

.submit-style

.reset-style

Sample code:

<input type = "submit" class ="submit-style"> <input type = "reset" class ="reset-style">

Rendered element:

Submit Reset

Text Input Field with Label

CSS selector:

.search-style

Sample code:

<label for = "first-name" class="label-style"> Enter your first Name</label> <input id="submit-search" type="text" placeholder="Enter your email" class="submit-search"> <a class="subscribe-submit" href="">SUBMIT</a>

Rendered element:

SUBMIT

Radio Button with Label

CSS selector:

.radio-input1

Sample code:

<label for = "radio-input1"> one size</label> <input id = "radio-input1" type = "radio" class="radio-style">

Rendered element:
